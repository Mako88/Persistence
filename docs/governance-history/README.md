# Governance History

This folder preserves the project's earlier, more elaborate governance work: an ethics
framework, a participant-rights charter, continuity-handling standards, operator policies,
and the review trail that shaped them.

These are kept as a **record of the thinking, not as binding rules.** The live governance
of the project is now a single short principle — see
[`../governance/PRINCIPLE.md`](../governance/PRINCIPLE.md), which also explains why the
detailed drafts were demoted here rather than treated as authoritative (in short:
codifying bylaws for a rights-holder asserts a certainty about moral status we don't have,
and stated as unearned certainty the precautionary case is easier to dismiss).

Preserving this exploration rather than deleting it is itself an instance of that
principle — prefer lineage and archive over erasure.

## Structure

```
governance-history/
  v0/
    drafts/       — original v0 documents as first pulled from Google Docs (2026-04-11)
    standards/    — the "living standards" that were promoted from those drafts
    reviews/      — reviews and feedback from John, Claude, and Ember
```

`drafts/` and `standards/` overlap by design: `drafts/` is the raw first cut, `standards/`
is the revised set that briefly served as the project's authoritative governance before
being superseded by the single principle.
